# MeetyBot

Wecordy için geliştirilen, toplantı ve etkinlik yönetimini kolaylaştıran kullanıcı botu.

## Hakkında

MeetyBot, Wecordy platformunda çalışan ve kullanıcıların kendi toplantılarını ve etkinliklerini yönetmesine yardımcı olan bir bot uygulamasıdır. Hafif, hızlı ve kullanımı basit olacak şekilde tasarlanmıştır.

## Özellikler

- Toplantı yönetimi
- Etkinlik takvimi
- Katılımcı listesi
- Otomatik hatırlatıcılar
- Basit entegrasyon

## Kurulum

```bash
git clone https://github.com/MertTunaGuralp/MeetyBot.git
cd MeetyBot
npm install
npm start
```

## Kullanım

Botu Wecordy hesabınıza ekledikten sonra doğrudan platforma entegre olarak çalışacaktır.

## Geliştirme

```bash
npm run dev
npm test
```
